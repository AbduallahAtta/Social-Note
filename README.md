The application is built using Google Firebase technologies, Android JetPack, Koin, and a set of third-party libraries with MVVM design pattern.

**Features**
* Rich Editor
* CRUD operations on notes
* Note time -reminder
* Note Location "Geofence" reminder
* Note-Sharing functionality "Posts"
* Social interact between users "Like & Comment"
* Friendly UI
